What if you disappear too?

The best way to avoid any future outages for packs and servers is to leverage a DNS CNAME. Combined with the pending backup solution, should this server ever disappear, you can fire up your own, import the backup and just update your DNS setting and no one will be the wiser.

To set up a CNAME to this server, in your DNS add the following:

Record Type CNAME

Value whitelist.yourdomain.com

TTL 3600

Target minecraft-whitelist.wearelucida.com

With this complete, you can use http://whitelist.yourdomain.com/yourtwitchid/whitelist.txt (and security key if applicable) which will be future-proof™.
